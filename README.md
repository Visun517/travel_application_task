# Simple Travel Application 🌍

This project is a structured Flutter application developed as an internship assessment. [cite_start]The goal is to build a user-friendly travel app while demonstrating clean UI development, efficient data handling, and proper navigation practices[cite: 5, 6].

## ✨ Features & Implementation

### [cite_start]1. Home Screen [cite: 13]
- [cite_start]**Banner UI:** Features a top banner image to provide a premium look[cite: 14].
- [cite_start]**Destinations List:** Displays a scrollable list of travel destinations using `ListView.builder`[cite: 15].
- [cite_start]**Place Cards:** Each location is showcased in a custom card with its title, location, and rating[cite: 16].

### [cite_start]2. Details Screen [cite: 17]
- [cite_start]**Immersive Header:** Displays a high-quality image of the selected destination[cite: 18].
- [cite_start]**Information:** Includes the title, a detailed "About" description, and a dynamic rating section[cite: 19].
- [cite_start]**Navigation:** Includes a back button for easy navigation[cite: 17].

### [cite_start]3. Profile Screen [cite: 20]
- [cite_start]**User Identity:** Displays the user's name, email, and a circular profile image[cite: 21, 22].
- **Activity Stats:** A clean section to view user statistics (Trips, Favorites, and Reviews).
- **Settings Menu:** A list of interactive menu items for account management.

### [cite_start]4. Navigation & Data [cite: 34, 37]
- [cite_start]**Bottom Navigation Bar:** Includes items for Home, Favorites, and Profile screens[cite: 25].
- [cite_start]**Data Modeling:** Uses a structured `Place` model to handle destination data[cite: 38].
- [cite_start]**Dynamic Passing:** Transfers data between screens to show specific details for each place[cite: 39].

## 🛠️ Tech Stack & Practices
